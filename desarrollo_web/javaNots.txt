

Pasos para inicializar entorno:
Anters quer nada debes de estar en la caroeta del proyecto
1. Inicializar un nuevo proyecto de Node.js:  
    npm init -y

    Este comando crea un archivo package.json en tu directorio, el cual gestiona las dependencias y scripts de tu proyecto.
2. Instalar express:
    npm install express

    instala express como una dependencia en el proyecto, lo que permite ultilizarlo para construir el servidor

4. Modificar package.json para soportar ES6:
    "type": "module"

    agregar esto permite el uso de la sintaxis  import/export de ES6.

5.Ejecuta el servidor:
    node app.js (nombre de tu archivo)

Comandos curl para interactuar con el servidor de cartas

1. Obtener todas las cartas:
    curl http://localhost:3000/cartas

2. Obtener una carta por ID:
    curl http://localhost:3000/cartas/id  (id solo es el numnero del id)

3. Agregar cartas:
    curl -X POST http://localhost:3000/cartas -H "Content-Type: application/json" -d '{"id": "  {id}", "nombre": "{nombre}", "atributo": "{atributo}"}'

4. Borrar una carta por ID:
    curl -X DELETE http://localhost:3000/cartas/{id}

5. Actualizar una carta por ID:
    curl -X PUT http://localhost:3000/cartas/{id} -H "Content-Type: application/json" -d '  {"nombre": "{nuevoNombre}", "atributo": "{nuevoAtributo}"}'
